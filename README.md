# Yatube API

Проект по API для работы с постами.
Можно комментировать посты и подписываться на автора.

### Функциональные возможности
#### API предоставляет следующие возможности:

- Управление постами: создание, чтение, обновление и удаление записей.
- Работа с комментариями: добавление, просмотр, редактирование и удаление комментариев к постам.
- Сообщества: просмотр списка сообществ и детальной информации о них.
- одписки: управление подписками на авторов (создание и просмотр списка).
- Аутентификация: получение, обновление и проверка JWT‑токена для авторизации.

### Как запустить проект:

##### Cоздать и активировать виртуальное окружение:
```
python -m venv venv
source venv/Scripts/activate
```
##### Установить зависимости из файла requirements.txt:
```
pip install -r requirements.txt
python -m pip install --upgrade pip
```

##### Выполнить миграции:
```
python manage.py migrate
```
##### Запустить проект:
```
python manage.py runserver
```
